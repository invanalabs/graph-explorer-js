# graph-explorer-js

Javascript library to build data graph using GraphSON data.


## Usage
```javascript
import React from "react";
import GraphExplorer from "./graph-explorer/graph-explorer";

export default class App extends React.Component {

    render() {
        return (
            <div className="App">
                <GraphExplorer/>
            </div>
        );
    }
}
```
